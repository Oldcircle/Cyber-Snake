**Neon Cyber-Snake** is a visually stunning, cyberpunk-themed arcade game built with React and Tailwind CSS. It features smooth controls, particle effects, and a complete dashboard for customization.

### Features

*   **Cyberpunk Aesthetic**: Glowing neon visuals, glassmorphism UI, and dynamic animations.
*   **Dual Language**: Full support for **English** and **Chinese**.
*   **Dashboard**:
    *   **Skin Selector**: Choose from 4 unique snake skins (Cyber, Gold, Matrix, Flame).
    *   **Map Themes**: Switch between different background atmospheres (Deep Space, Retro, Clean).
    *   **History**: Tracks your top scores with timestamps.
*   **Gameplay Mechanics**:
    *   Classic Snake logic.
    *   **Dash Mechanic**: Hold `Space` to speed up (High risk, high reward).
    *   Responsive controls (WASD or Arrow Keys).
*   **Tech Stack**: React 18, Tailwind CSS, TypeScript (No build tools required).

### How to Play

1.  Open `index.html` in any modern browser.
2.  **Controls**:
    *   `â†‘` `â†“` `â†` `â†’` or `W` `A` `S` `D`: Move.
    *   `Space`: Dash (Speed up).
    *   `Esc` or `P`: Pause.
3.  Eat the glowing nodes to grow and score points. Avoid walls and your own tail.

### Customization

Click the **DASHBOARD** button in the main menu to access settings. Your preferences (Language, Skin, Theme) and high scores are saved automatically to your browser's LocalStorage.
